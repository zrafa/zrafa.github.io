Bienvenidos

Este es el sitio web del proyecto de investigación 04/F005: Identificación, Evaluación y Adaptación de Componentes de Hardware y Software para Sistemas Embebidos, perteneciente a la Facultad de Informática de la Universidad Nacional del Comahue. En este espacio se puede acceder a información relacionada a todos los subproyectos, artículos y documentación del proyecto.
